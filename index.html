Cyber Security Research

Oregon State University

Recently finished a project under Dr. Attila Yavuz to implement and analyze methods an adversary would use to capture and exfiltrate large quantities of data from IoT devices. This consisted of implementing a metamorphic agent which can alter its executable signature, gather keystrokes, capture images, encrypt the captured data, send the data to a centralized server, and remove its trace from the infected system. The controller utilized a weak exploit to target and execute the agent on a large quantity of vulnerable IoT devices to maximize spreading potential. Static analysis software developed by Microsoft, McAfee, Avast, and 55 others were ineffective at identifying the agent. Some potential defensive mechanisms were investigated including an automated approach to generate common malware behavior patterns for detection of metamorphic malware or new malware instances by combining static analysis and data-mining techniques.
